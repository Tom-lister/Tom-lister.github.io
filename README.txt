To preview on your computer (not yet working for mobile):

1. Open command line
2. Type 'node -v' to check if Node.js is installed.
3. If not, install here: https://nodejs.org/en/download

5. Still in command line, navigate into folder containing this file (e.g. 'cd C:\Users\tomlis\OneDrive\Dokumente\town-square')
6. Type 'npm install'
7. Type 'npm run dev'

8. Open the displayed URL
9. Press F12 in chrome to open developer tools and switch to mobile view (upper left button)
10. Edit the contents of 'Posts.tsx' to change which posts appear, adding any new images to the appropriate folders in 'public'

To terminate the server, return to the command line and type ctrl-C
